第一次設定

第一次發布需要貼 GitHub 權杖。每台新電腦第一次使用後台時都要貼一次，貼上後那台電腦會記住；網站內容本身會存在 GitHub 雲端資料裡。

建立權杖

權杖建議設定：只給 yunglin10/po 這個 repo 的 Contents: Read and write。

進階功能

平常不用這裡。需要備份或嵌入 Google Sites 時再打開。

裁剪圖片

拖曳圖片調整位置，使用滑桿縮放，確認後會存成適合網頁使用的圖片。

小提醒：首頁背景建議用 16:9；創辦人照片建議 4:5；施工案例通常用 4:3 最穩。
