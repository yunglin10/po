完全免費。第一次使用要先建立 GitHub 權杖，權杖請只貼在這個欄位，不要傳給任何人。

建立權杖 開啟正式網站

權杖建議設定：只給 yunglin10/po 這個 repo 的 Contents: Read and write。發布後通常 1 到 3 分鐘網站會更新。

3. Google Sites 嵌入碼

按下產生後，可以複製整段嵌入碼到 Google Sites 的「嵌入」區塊。嵌入碼很長，正常。

裁剪圖片
